### **Navigation**

Yuko Dashboard → **Settings → Reviews → Branding**

![bran.png](/images/bran.png)

### **Overview**

Personalize how your reviews appear across all widgets and emails.\
Choose your preferred color palette, font, and star icon style to match your store’s unique branding.\
All visual elements — from button colors to text styles — can be customized for a consistent brand experience.

### **Branding Options**

| **Setting**                      | **Description**                                                                                                                                         |
| :------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Color Palette**                | Choose from preset themes such as _Eclipse Black, Coral Pink, Herbal Green, Golden Ochre, Crimson Red, Espresso Brown, Pacific Blue,_ or _Modern Gray._ |
| **Primary Color**                | The main accent color used for stars, buttons, and highlights.                                                                                          |
| **Secondary Color**              | A softer accent color used for backgrounds or secondary elements.                                                                                       |
| **Primary Text Color**           | Default color for review titles and main text.                                                                                                          |
| **Secondary Text Color**         | Used for less prominent text, such as timestamps or meta details.                                                                                       |
| **Icon Color**                   | Defines the color of icons such as the star rating.                                                                                                     |
| **Review Font**                  | Choose the font family (e.g., Arial) used to display reviews in widgets.                                                                                |
| **Star Rating Icon**             | Select the star icon style — _Star-Sharp, Star-Rounded,_ or _Classic._                                                                                  |
| **Apply Changes to All Widgets** | Apply the same color and font settings across all review widgets at once.                                                                               |
| **Save Changes**                 | Save your new branding preferences.                                                                                                                     |
