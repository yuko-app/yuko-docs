**Navigation:**

Yuko Dashboard → Reviews → Review Collection Form → (choose Email or On-site)

### **Overview**

You can customize review input fields such as rating, photos, title, review content, and reviewer details.\
 These inputs define how customers interact with your form both in **review request emails** and on your **storefront**.

### **Settings**

| **Section**        | **Description**                                                                |
| :----------------- | :----------------------------------------------------------------------------- |
| **Rating**         | Choose the number of stars and display text (e.g., _Dislike it_ → _Love it!_). |
| **Photos**         | Allow customers to upload up to 5 photos per review to improve authenticity.   |
| **Review Content** | Add optional or required fields for review title and detailed feedback.        |
| **Reviewer Info**  | Collect details like reviewer name and email — useful for verification.        |
| **Thank You**      | Personalize the post-submission confirmation message.                          |
| **Next Product**   | Enable follow-up review requests for other purchased items.                    |
| **Footer**         | Add disclaimers, terms, or “Powered by Yuko” text.                             |
| **Advanced CSS**   | Customize form design using your own CSS for perfect brand alignment.          |
