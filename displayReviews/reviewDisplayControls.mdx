### **Navigation**

Yuko Dashboard → **Settings → Reviews → Review Display Controls**

### **Overview**

Yuko gives you granular control over how reviews are displayed across widgets and product pages.

### **Settings**

| **Setting**               | **Description**                                              |
| :------------------------ | :----------------------------------------------------------- |
| **Reviewer Name Format**  | Display first name only, last name only, or both.            |
| **Date Format**           | Customize the appearance of review dates.                    |
| **Sorting Preference**    | Default order for displayed reviews (Newest, Highest, etc.). |
| **Empty States**          | Customize the message shown when no reviews are available.   |
| **Pagination**            | Choose between static pages or infinite scrolling.           |
| **Cross-product Display** | Show related reviews if a product has few submissions.       |
