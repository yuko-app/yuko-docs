### **Navigation**

Yuko Dashboard → **Settings → Integrations → Webhooks**

![wwe.png](/images/wwe.png)

### **Overview**

Use **Webhooks** to send Yuko reviews and order data to external systems in real time.\
This is ideal for custom integrations or connecting to tools that don’t have native Yuko apps.

### **Setup Steps**

1. Go to **Settings → Integrations → Webhooks**.
2. Click **Add Webhook**.
3. Enter your **Webhook URL**.
4. Select the **events** you want to send:
   - Review Submitted
   - Review Approved
   - Review Replied
   - Order Completed

Save your webhook.

### **Example Use Case**

Send new review data to your CRM, analytics tool, or notification system instantly when a review is approved.
