### **Navigation**

Yuko Dashboard → **Settings → Integrations → Zapier**

### **Overview**

Connect Yuko with thousands of apps through **Zapier** — no code required.\
Automate tasks between Yuko and tools like Slack, Google Sheets, Mailchimp, Klaviyo, and more.

### **Setup Steps**

1. Go to **Settings → Integrations → Zapier**.
2. Click **Generate API Key** (if you haven’t already).
3. Log in to **Zapier** and search for **Yuko**.
4. Choose a trigger event (e.g., “New Review Submitted”).
5. Choose an action app (e.g., “Add Row in Google Sheets”).

Test and activate your Zap.

## **Popular Use Cases**

| **Use Case**      | **Description**                                       |
| :---------------- | :---------------------------------------------------- |
| Slack Alerts      | Send instant Slack messages when new reviews arrive.  |
| Google Sheets Log | Record all reviews in a shared sheet automatically.   |
| Email Marketing   | Add reviewers to a special “Reviewed Customers” list. |
