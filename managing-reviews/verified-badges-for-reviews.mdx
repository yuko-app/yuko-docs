Showcase authenticity by displaying “Verified Buyer” badges beside legitimate reviews.

**Requirements:**

- The reviewer must have purchased the product via your connected store (WooCommerce or Shopify).
- The review must be tied to an order in Yuko.

**How to Add a Verified Badge**

1. Navigate to **Manage Reviews** from the top menu.
2. Find the review you want to verify.
3. Click the **ellipsis icon (⋮)** on the right side of the review.
4. Select **Add Verified Badge** from the dropdown options.
5. The **Verified Buyer** label will instantly appear beside the reviewer’s name.

   ![Addvb Pn](/images/addvb.png)

Once enabled, Yuko automatically adds the badge to all qualifying reviews.

### **Review Display Controls**

Control how reviews appear across your store.

| **Setting**                                   | **Description**                                                                                                          |
| :-------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------- |
| **Reviewer Name Format**                      | Choose how the reviewer’s name appears on widgets — full name, first name, or initials.                                  |
| **Date Format**                               | Select your preferred date display format for reviews (e.g., YYYY-MM-DD).                                                |
| **Choose How Products Are Shown for Reviews** | Decide the order in which products appear in review request emails and widgets (e.g., Most Expensive, Cheapest, Random). |
| **Number of Review Inputs**                   | Set how many products are shown in review request emails for customers to review (maximum: 5).                           |
| **Add Review Button Text**                    | Customize the text displayed on the button that allows customers to submit new reviews.                                  |

**Navigation:**

\
Yuko Dashboard → **Settings → Reviews → Review Display Controls**
