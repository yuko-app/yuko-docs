### **Navigation**

Yuko Dashboard → **Reviews → Widgets → All Reviews**

![All Pn](/images/all.png)

The Wall of Fame Widget displays every approved review from your store — across all products — in one clean, organized layout.\
It’s ideal for testimonial pages, community pages, landing pages, or any page where you want to show social proof at scale.

This widget helps customers explore real experiences from your entire store, not just one product, which can greatly improve trust and conversions.

## How It Works

The Wall of Fame Widget collects **all approved reviews** (product reviews, shop reviews, or both) and displays them based on your selected layout.

You can customize:

- Which types of reviews appear
- Whether reviews with photos/videos appear first
- How reviews load (all at once or infinite scroll)
- Headings and labels
- Colors, spacing, and typography

This allows you to design a review wall that fits your brand and page style.

### 3. Preferences

| Option                | Description                                                               |
| :-------------------- | :------------------------------------------------------------------------ |
| Display               | Show or hide the widget                                                   |
| Show Review Title     | ON/OFF                                                                    |
| Show Review Date      | ON/OFF                                                                    |
| Show Product Variants | ON/OFF                                                                    |
| Show Product Block    | ON/OFF                                                                    |
| Header Section        | Show main header section                                                  |
| Out of / Based on     | Rating text labels                                                        |
| Verified Badge Text   | Customize verified badge text                                             |
| Verified Buyer        | Display verified buyer info                                               |
| Reply Section Text    | Customize reply header text                                               |
| Replied!              | Text for replied reviews                                                  |
| No Reviews Title      | Empty state title when no reviews exist                                   |
| Button Text           | Pagination button text                                                    |
| Tab Section           | Reviews tab text                                                          |
| Sort Options Text     | Label for sorting options (Newest, Oldest, Highest Rating, Lowest Rating) |
| Sorting               | Enable/disable sorting options                                            |
| Reviews Per Page      | Number of reviews per page                                                |
| Filtering             | Enable/disable rating filter options                                      |

> **Note:** Changes can be previewed on your store by clicking **Check store to view this change**.
