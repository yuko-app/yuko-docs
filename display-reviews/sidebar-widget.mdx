### **Navigation**

Yuko Dashboard → **Reviews → Widgets → Reviews Sidebar**

![NEXT Pn](/images/NEXT.png)

### **Overview**

The **Sidebar Widget** is a floating tab that reveals customer reviews from any page.\
It’s a great way to display reviews without taking up the main content space.

### **Settings**

| **Setting**        | **Description**                                                                            |
| :----------------- | :----------------------------------------------------------------------------------------- |
| **Position**       | Left or Right side of the screen.                                                          |
| **Orientation**    | Reviews scroll _Top → Bottom_ or _Bottom → Top_.                                           |
| **Button Text**    | Customize the tab label (e.g., “Customer Reviews”).                                        |
| **Content Source** | Choose to display the current product’s reviews, related reviews, or all reviews globally. |
| **Appearance**     | Adjust colors for background, stars, and text.                                             |
| **Responsive**     | Option to hide widget on mobile devices.                                                   |
